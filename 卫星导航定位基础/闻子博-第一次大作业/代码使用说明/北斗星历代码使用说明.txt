 代码功能概述
此代码主要用于处理北斗卫星的星历数据，实现从读取原始星历文件，提取北斗卫星数据，将数据存储为新文件，再将数据解析为字典格式，最后计算卫星位置等一系列操作，并将结果保存到 CSV 文件中。

代码使用说明

 1. 环境准备
确保你已经安装了 Python 环境，并且安装了所需的库：
```bash
pip install numpy
```

2. 数据准备
- 确保你有一个北斗卫星的星历文件，文件格式为 `RNX`，例如 `BRDC00GOP_R_20250590000_01D_MN.rnx`。
- 将星历文件的路径修改为你实际文件的路径，代码中默认的文件路径为：
```python
infile = open(r"D:\python\BRDC00GOP_R_20250590000_01D_MN.rnx\BRDC00GOP_R_20250590000_01D_MN.rnx", "r")
```
请将上述路径修改为你实际的文件路径。

3. 代码运行步骤

3.1 提取北斗卫星数据
代码会自动读取原星历文件，并提取其中的北斗卫星数据，将其保存为 `all_C.txt` 文件。

```python
infile = open(r"D:\python\BRDC00GOP_R_20250590000_01D_MN.rnx\BRDC00GOP_R_20250590000_01D_MN.rnx", "r")
content = infile.readlines()
infile.close()
start_num = 0
all_C = []  # 存储北斗星历数据块
for i in range(len(content)):
    if content[i].find("END OF HEADER") != -1:
        start_num = i + 1
for i in range(len(content) - start_num):
    line = content[start_num + i]
    if line[0] == 'C':
        for j in range(0, 8):
            all_C.append(content[start_num + i + j])
outFile = open("all_C.txt", 'w')  # 存储为新的文件（只含有北斗卫星）
outFile.writelines(all_C)
outFile.close()
```

 4. 注意事项
- **`start_num` 函数**：代码中定义了 `start_num` 函数，但函数体为空，需要根据实际情况实现该函数，以确定数据记录的起始行。
- **`rx` 和 `rz` 函数**：代码中定义了 `rx` 和 `rz` 函数，但函数体为空，如果你需要使用这两个函数，需要根据实际情况实现它们。
- **观测时间 `t1`**：代码中 `t1` 变量赋值为 `None`，并且在计算归化时间 `t_k` 时，由于没有接收机的观测时间，为了练手设 `t_k = -14`，实际使用时需要根据接收机接收到的时间进行计算。
