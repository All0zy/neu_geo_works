### 代码功能概述
这段Python代码主要用于读取卫星误差分析数据文件，并绘制不同卫星的平均百分差、最大百分差、最小百分差和标准差百分差的柱状图。

### 代码结构
代码主要包含以下几个部分：
1. **导入必要的库**：导入`pandas`和`matplotlib.pyplot`库。
2. **读取卫星数据函数**：`read_satellite_data`函数用于读取指定文件中的卫星误差数据，并将其转换为`pandas`的`DataFrame`对象。
3. **绘制百分差柱状图函数**：`plot_percentage_differences`函数用于根据`DataFrame`对象中的数据绘制柱状图。
4. **主程序**：在`if __name__ == "__main__":`块中，调用上述两个函数完成数据读取和绘图操作。

### 使用步骤

#### 1. 安装依赖库
在运行代码之前，需要确保已经安装了`pandas`和`matplotlib`库。可以使用以下命令进行安装：
```bash
pip install pandas matplotlib
```

#### 2. 准备数据文件
数据文件应遵循特定的格式，示例如下：
```
卫星1:
平均百分差: 1.2
最大百分差: 3.4
最小百分差: 0.5
标准差百分差: 0.8
卫星2:
平均百分差: 2.1
最大百分差: 4.3
最小百分差: 1.0
标准差百分差: 1.2
```
每颗卫星的数据以卫星名称开头，后面跟着冒号。每个百分差数据以相应的名称开头，后面跟着冒号和具体数值，数值可以包含百分号。

#### 3. 修改文件路径
在代码的主程序部分，需要修改`file_path`变量的值，将其设置为实际的数据文件路径。例如：
```python
if __name__ == "__main__":
    file_path = r'C:\Users\YourUsername\Desktop\误差分析.txt'
    satellite_df = read_satellite_data(file_path)
    plot_percentage_differences(satellite_df)
```

#### 4. 运行代码
将修改后的代码保存为一个Python文件（例如`误差图.py`），然后在命令行中运行该文件：
```bash
python 误差图.py
```

#### 5. 查看结果
运行代码后，会弹出一个窗口显示绘制好的柱状图，图中展示了不同卫星的平均百分差、最大百分差、最小百分差和标准差百分差的分布情况。

### 注意事项
- 数据文件的编码格式应为UTF-8。
- 确保数据文件中的百分差数值为有效的浮点数，否则可能会导致代码运行出错。
- 代码中的柱状图宽度、颜色等样式可以根据需要进行调整，修改`plot_percentage_differences`函数中的相关参数即可。
