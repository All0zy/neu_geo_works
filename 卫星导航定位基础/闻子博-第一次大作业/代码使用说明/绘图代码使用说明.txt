### 代码使用说明

#### 1. 代码功能概述
此代码的主要功能是读取包含北斗卫星坐标信息的文本文件，并将这些坐标数据以二维和三维视图的形式进行可视化展示。用户可以通过鼠标悬停在散点上查看每个卫星的详细信息，包括卫星编号和对应的 X、Y、Z 坐标。

#### 2. 环境准备
在运行代码之前，需要确保已经安装了以下 Python 库：
- `matplotlib`：用于绘制图形和可视化数据。
- `mplcursors`：用于实现鼠标悬停提示功能。

可以使用以下命令来安装这些库：
```bash
pip install matplotlib mplcursors
```

#### 3. 数据文件准备
代码需要一个文本文件来读取北斗卫星的坐标信息，文件名为 `北斗卫星位置.txt`。该文件的格式要求如下：
- 每行以 `C` 开头，表示卫星编号。
- 每行包含至少四个用逗号分隔的部分，分别为卫星编号、X 坐标、Y 坐标和 Z 坐标。

示例文件内容：
```
C01,100,200,300
C02,400,500,600
```

#### 4. 代码结构和主要函数说明
- **`read_bds_coordinates(filename)`**：
  - 功能：从指定的文本文件中读取北斗卫星的坐标信息。
  - 参数：`filename` - 包含卫星坐标信息的文本文件的名称。
  - 返回值：返回四个列表，分别为卫星编号列表 `ids`、X 坐标列表 `x`、Y 坐标列表 `y` 和 Z 坐标列表 `z`。

- **`plot_bds_coordinates(ids, x, y, z)`**：
  - 功能：根据读取的卫星坐标信息绘制二维和三维视图，并添加鼠标悬停提示功能。
  - 参数：
    - `ids`：卫星编号列表。
    - `x`：X 坐标列表。
    - `y`：Y 坐标列表。
    - `z`：Z 坐标列表。
  - 返回值：无，直接显示绘制的图形。

#### 5. 代码运行步骤
1. 将包含卫星坐标信息的文本文件命名为 `北斗卫星位置.txt`，并确保该文件与代码文件 `绘图.py` 位于同一目录下。
2. 打开终端或命令提示符，进入代码文件所在的目录。
3. 运行以下命令来执行代码：
```bash
python 绘图.py
```
4. 代码运行后，会弹出一个窗口，显示二维和三维视图的卫星坐标散点图。
5. 将鼠标悬停在散点上，可以查看每个卫星的详细信息，包括卫星编号和对应的 X、Y、Z 坐标。

#### 6. 注意事项
- 确保数据文件 `北斗卫星位置.txt` 的格式符合要求，否则可能会导致数据读取错误。
- 如果需要修改数据文件的名称或路径，可以在代码中修改 `read_bds_coordinates` 函数的参数。
- 可以根据需要调整图形的大小、颜色、视角等参数，具体可在 `plot_bds_coordinates` 函数中进行修改。
