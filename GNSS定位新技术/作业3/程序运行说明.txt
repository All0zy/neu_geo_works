一、程序运行环境

操作系统:
  - Windows 11

Python 版本:
  - 推荐 Python 3.10 或 3.11
  
IDE/编辑器:
  - Visual Studio Code

二、依赖的库

核心科学计算库:
  - numpy         (>=1.21.0)      # 数值计算
  - pandas        (>=1.3.0)       # 数据处理和分析
  - scipy         (>=1.7.0)       # 科学计算(插值、信号处理)
  - matplotlib    (>=3.4.0)       # 绘图

地理和几何处理:
  - shapely       (>=1.7.0)       # 地理边界和几何操作
  - pyproj        (可选)          # 坐标系转换

其他库:
  - imageio       (>=2.12.0)      # 图像处理(生成动画)

三、运行方式

【1】运行 PPP 批处理 (hkcl_batch_ppp.py)
功能: 对 hkcl 目录下的观测文件进行 PPP 精密单点定位处理

【2】运行 VTEC 处理 (main.py)
功能: 处理 RINEX 观测文件并计算电离层 VTEC 数据

【3】运行 hkcl VTEC 处理 (hkcl_vtec_no_interp.py)
功能: 处理 hkcl 目录下的观测文件计算 VTEC (无插值热力图)

【4】运行混合格式 VTEC 处理 (obv_vtec_mixed_rinex.py)
功能: 处理 obv/ 目录中混合 RINEX 格式的观测文件

四、输入数据

【观测数据 - RINEX 格式】

文件位置:
  - hkcl/        # 香港测站观测文件
  - obv/         # 其他观测文件
  - 9obv/        # 其他观测文件集合

观测数据类型:
  - 伪距 (PR1, PR2, PR5等)
  - 载波相位 (LLI, CP1, CP2, CP5等)
  - 信号强度 (CNR)
  - 支持 GPS/GLONASS/Galileo/BeiDou 多制式


【轨道/钟差文件】

导航星历 (RINEX NAV):
精密轨道 (SP3):
精密钟差 (CLK):
偏差文件 (BIA):
天线文件 (ATX)


【边界数据】

地理边界文件:
  香港特别行政区.geojson

五、输出数据

【PPP 处理输出 (hkcl_batch_ppp.py)】

输出目录:
  output/hkcl_ppp_batch/

每日输出文件:
  1. HKCL_YYYDOY_ppp_results.csv
     - 每个历元的定位结果

  2. HKCL_YYYDOY_position_error.png
     - 位置误差时间序列图 + ZTD 变化
     - 显示不同定位模式的精度

  3. HKCL_YYYDOY_error_distribution.png
     - 东/北/天向误差的直方图分布

  4. HKCL_YYYDOY_3d_error.png
     - 三维误差散点云

  5. HKCL_YYYDOY_accuracy_analysis.png
     - 水平精度随时间变化曲线

汇总统计输出:
  HKCL_7days_ppp_summary.csv

【VTEC 处理输出 (main.py / hkcl_vtec_no_interp.py)】

输出目录:
  output/vtec_results/ (或 output/hkcl_vtec/)

每个观测文件输出:
  1. {station}_vtec.csv
     - 时间, VTEC 值, 覆盖卫星数, 电离层穿刺点信息

  2. {station}_vtec_analysis.png
     - 4 个子图:
       a) VTEC 时间序列 (彩色表示观测方向)
       b) VTEC 空间分布 (极坐标)
       c) 卫星可见性分析
       d) VTEC 统计直方图

  3. vtec_comparison_overlay.png
     - 多个站点的 VTEC 对比叠加图

六、工作目录结构说明

根目录:
  ├── cssrlib/                    # 核心库文件
  │   ├── gnss.py                # GNSS 基础函数
  │   ├── rinex.py               # RINEX 文件处理
  │   ├── peph.py                # 精密轨道/偏差处理
  │   ├── ppp.py                 # PPP 算法
  │   ├── pppssr.py              # PPP + SSR 集成
  │   ├── ephemeris.py           # 星历计算
  │   ├── mlambda.py             # 模糊度解算
  │   └── cssrlib.py             # 通用工具
  │
  ├── hkcl/                       # 香港测站数据 (输入)
  │   ├── HKCL*.26o              # RINEX 观测文件
  │   ├── BRDC*.rnx              # 导航电文
  │   ├── *.SP3                  # 精密轨道
  │   ├── *.CLK                  # 精密钟差
  │   ├── *.BIA                  # 观测偏差
  │   ├── M14.ATX, M20.ATX, I20.ATX  # 天线文件
  │   └── ...
  │
  ├── obv/, 9obv/                # 其他观测文件 (输入)
  ├── sp3/                        # SP3 轨道文件 (输入)
  │
  ├── output/                     # 处理输出结果
  │   ├── hkcl_ppp_batch/        # PPP 结果
  │   ├── hkcl_vtec/             # HKCL VTEC 结果
  │   ├── vtec_results/          # 通用 VTEC 结果
  │   └── ...
  │
  ├── hkcl_batch_ppp.py          # PPP 批处理脚本 (主程序1)
  ├── main.py                     # VTEC 处理脚本 (主程序2)
  ├── hkcl_vtec_no_interp.py     # HKCL VTEC (主程序3)
  ├── obv_vtec_mixed_rinex.py    # 混合格式 VTEC (主程序4)
  ├── 香港特别行政区.geojson       # 地理边界数据
